## Overview

Image generation prompts for all five avatar emblems. Reference art generated and approved via Midjourney. All five locked and ready for SVG trace.

These prompts are archived here for future iteration use — additional variants, Lottie reference generation, scale testing.

---

## How to Use

**Midjourney:** Copy the MJ prompt exactly. The suffix `--style raw --no color --ar 1:1` is locked for all five. Do not change it. Run 4 variations, select strongest geometry, upscale.

**Gemini:** Use the Gemini prompt — descriptive-first since Gemini responds to natural language over technical parameters. If results are too decorative add: *"Keep the design minimal and geometric. No decorative flourishes."*

**What to look for in a good result:**
- Clean circular boundary (except Ignition — intentional break at apex is correct)
- Consistent stroke weight — nodes and lines feel like they belong to the same system
- Readable at thumbnail size — if it blurs at small scale, reject it
- Monochrome only — black on white, no baked-in colour
- No background noise, no environment, no glow effects

**After generation:** A designer traces the best result into a clean SVG. The prompt output is reference art, not the final asset.

## Shared Design Rules

- Circular boundary on all — except Ignition where the break is intentional
- Circuit trace aesthetic throughout — consistent stroke weight and node size
- Node style: hollow circles, identical size across all nodes within one emblem
- Must read clearly at 24px — this is the hard production constraint
- Monochrome base — colour applied as a layer on top, never baked in
- No fill, no gradient, no decorative detail
