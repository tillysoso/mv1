**Version 1.0 — Living Document**

## What This Document Is

The Majestic brand guidelines are the single source of truth for anyone building, writing, designing, or communicating on behalf of Majestic. This document covers brand identity, colour, typography, voice, and the rules that protect the integrity of everything we have built.

It is a living document. It will grow as the product grows. Version history should be maintained.

---

## 01 — Brand Foundation

### What Majestic is

Majestic is an intuition coaching tool that uses the ancient symbolic system of tarot to help users shift their thinking and trust their gut instincts.

It is not a fortune-telling app. It is not a wellness platform. It is not a personality test.

### Who it is for

People who think in systems, escape into worlds, and have spent years trusting everyone else's read of a situation. They overlap across three subcultures — anime-spiritual, tabletop-lore, and occult-esoteric — but they share one trait: they are exceptionally good at reading patterns in fictional systems and significantly less practised at applying that same intelligence inward.

### Why it exists

Majestic was built from a real story. A man who had presence, charisma, and decades of professional achievement — and quietly, privately, did not listen to himself for most of it. Until he did. And everything changed. That experience is what Majestic passes forward.

### The core promise

The avatars will be there. Majestic will always be there. But the insight and the work belongs to the user.

## 02 — Brand Line

### Primary line

**Your adventure. But Majestic.**

This is not a call to action. It is an invitation. It assumes the user already has a story, already has instincts, already has something worth trusting. Majestic is not the answer. It is the quality that was always possible inside their own journey.

### Flex variations

Use these across different touchpoints where the primary line needs specificity:

- Your next move. But Majestic.
- Your instincts. But Majestic.
- Your read. But Majestic.
- Your decision. But Majestic.

### Sign-off line

**You trusted yourself. Remember this.**

Used at the end of significant moments — post-reading, milestone acknowledgement, onboarding completion.

### Rules for the brand line

- Always sentence case — never all caps
- Never altered beyond the flex variations above
- Never used ironically
- The word Majestic in the line is always the brand name — never a descriptor

## 03 — Logo & Wordmark

### The wordmark

Luke's chosen script font. Warm, personal, handwritten energy. This is non-negotiable and locked.

**Usage rules:**
- Used for the brand name only — never for any other copy
- Minimum size: 80px wide on screen, 25mm in print
- Clear space: equal to the height of the capital M on all sides
- Never stretched, distorted, rotated, or redrawn
- Never used in avatar accent colours — only in Majestic purple or bone white

### Colour versions

| Version | Use case |
|---------|----------|
| Majestic purple `#9500FF` on dark | Primary — app, dark marketing surfaces |
| Bone white `#F0EDE8` on purple | Reversed — purple background surfaces |
| Charcoal `#1E1E2E` on light | Light backgrounds only |

### What the wordmark never does

- Appears in avatar accent colours
- Appears on backgrounds that reduce contrast below AA accessibility
- Appears at sizes below the minimum
- Appears in the Terminal or Cinzel fonts
- Is recreated or redrawn — always use the supplied file

### The master emblem

The Majestic convergence crest — the threshold gateway with four avatar node points. Used as:
- App icon
- Brand mark in contexts where the wordmark is too detailed
- Card back centrepiece
- Marketing emblem

Full rendering rules in Avatar Emblem System v1.0.

### Primary palette

| Name | Hex | Role |
|------|-----|------|
| Threshold deep | `#1A1A2E` | Primary background |
| Threshold mid | `#16213E` | Secondary surfaces |
| Threshold steel | `#0F3460` | Elevated surfaces |
| Bone white | `#F0EDE8` | Primary text |
| Mist | `#A8A8B8` | Secondary text, inactive states |

### Colour rules

**Do:**
- Use Majestic purple as the brand constant in all identity contexts
- Apply the gradient wash system for cards and portal surfaces
- Use elemental accents for interactive states and avatar-specific moments
- Maintain WCAG AA contrast ratios across all text and background combinations

**Never:**
- Use the elemental accents as background colours — they are always accents not foundations
- Use more than one elemental accent simultaneously unless showing the full system
- Use rainbow gradients or multiple saturated colours on a single surface
- Use pure flat black — always use the deep background neutrals with their blue undertone

## 05 — Typography

**Full type scale and implementation:** Typography System v1.0

### The system

| Font | Role | Never use for |
|------|------|---------------|
| Luke's script | Wordmark only | Any other copy purpose |
| Cinzel | Display, card titles, ceremonial moments | Body copy, navigation |
| Montserrat | Body, interface, avatar voice | Card titles, ceremonial display |
| Space Mono | Onboarding terminal sequence only | Any other context |

### Typography rules

**Do:**
- Track Cinzel generously — minimum 0.05em, typically 0.1–0.15em
- Keep body copy line heights generous — 1.6–1.7 for reading content
- Use Cinzel in all caps for Major Arcana card titles
- Use Montserrat Medium for avatar voice lines — the slight weight distinguishes voice from narration

**Never:**
- Use more than two fonts on a single screen
- Use Cinzel at body copy sizes
- Use all-caps for Montserrat body copy
- Use accent colours for running text
- Use the wordmark font for anything except the Majestic brand name

## 06 — Brand Voice

**Full voice personality and avatar voices:** Brand Voice Document

### How Majestic sounds

Warm but not soft. It has been through things. Direct but never prescriptive. It points — it does not push. Intelligent and comfortable with complexity without needing to show off. Celebratory without being loud. Comfortable with silence.

### What it never sounds like

- A wellness influencer
- A fortune teller
- A life coach with a framework to sell
- Someone performing wisdom
- Spiritual jargon or faux-archaic language
- Generic self-help positivity

### Voice by surface

| Surface | Register | Example |
|---------|----------|---------|
| App store and marketing | Scene-setting, not selling | *Some patterns take longer to read. The ones who look closer always find more.* |
| Onboarding | Warm, curious, unhurried | *You made it here for a reason. Let's begin.* |
| Card interpretations | Present-tense, emotionally intelligent | *There is something you already know about this situation.* |
| Notifications | Brief, atmospheric, never nagging | *Something worth reading is waiting for you.* |
| Empty states | World-building, never corporate | *The signal is finding you.* |

### The four avatar voices

Each avatar has a distinct communication style. Full voice profiles, sample lines, and what they never sound like are defined in the Brand Voice Document.

| Avatar | Voice mode | Never sounds like |
|--------|-----------|-------------------|
| Casper | Catalytic, direct, brief | Gentle affirmations or lingering reflections |
| Eli | Expansive, interpretive, idea-rich | Simple reassurance or practical step-by-step guidance |
| Olivia | Grounded, practical, clear | Abstract or poetic language |
| Destiny | Gentle, validating, reflective | Urgency or calls to action |

## 07 — Photography & Illustration

### Avatar illustration style

Loish painterly style — soft luminous rendering, emotionally expressive, world-made not heaven-made. Portrait orientation, 2:3 aspect ratio. Three states per avatar: neutral, active, reflective.

Full generation prompts in Avatar Midjourney Prompt Library v1.0.

### Card illustration style

Future-mythic eco-tech tarot. One central archetypal figure or object per card. Moonlit urban-nature setting, reflective water, botanical overgrowth integrated with architecture, restrained elemental glow. No text, no clutter, no medieval costumes.

Full prompt scaffold and constraints in Design Brief Handoff.

### The gradient wash

Luke's purple-to-element gradient wash is applied over all card illustrations as a photographic filter — edge-weighted, never obscuring the focal point. The Majestic purple base is always present at 8% opacity.

### What imagery never does

- Uses medieval European tarot iconography
- Features generic "purple galaxy" cosmic aesthetics
- Shows overt occult symbols — pentagrams, obvious mystical iconography
- Copies any specific anime franchise or game aesthetic
- Features perfect symmetry or pristine unearned beauty in avatars
- Uses pure white backgrounds

## 08 — The World

### Threshold City

The visual and narrative world of the entire app. Future-mythic luminous eco-tech folklore. All visual decisions should feel like they belong here.

When in doubt, ask: does this look like it exists in a near-future city where technology, botanical regrowth, and symbolic ritual coexist? If yes, proceed. If no, reconsider.

### The portal system

Every avatar appearance is mediated through a portal — an arch threshold between the user's world and Threshold City. The arch geometry echoes the master emblem. The card frame echoes the arch. One visual language across the entire product.

## 09 — What Majestic Is Not

These are active guardrails — not just things we avoid, but things we explicitly reject:

| Not this | Why |
|----------|-----|
| A fortune-telling app | We do not predict. We reflect. |
| A wellness platform | We respect intelligence. Affirmations are not our language. |
| Coded for one gender | The visual system, voice, and avatars are explicitly gender-neutral |
| A personality test | The quiz surfaces a suggestion. The user makes the choice. |
| A direct anime/tabletop imitation | The references live in the DNA, not the surface |
| A community or social platform | The work is private and personal |
| A replacement for professional support | We are a symbolic coaching tool — this is always acknowledged |

## 10 — Usage Do and Don't

### Logo and wordmark

| Do | Don't |
|----|-------|
| Use the supplied wordmark file | Recreate or redraw the wordmark |
| Use on approved background colours | Place on backgrounds that fail contrast |
| Maintain minimum clear space | Crowd the wordmark with other elements |
| Use Majestic purple or bone white versions | Use in avatar accent colours |

### Colour

| Do | Don't |
|----|-------|
| Lead with `#9500FF` in brand contexts | Replace the purple with any other brand colour |
| Use elemental accents as interactive signals | Use accents as background fills |
| Apply the gradient wash to cards and portals | Apply the wash as a flat colour block |
| Maintain WCAG AA on all text | Use accent colours for running text |

### Typography

| Do | Don't |
|----|-------|
| Track Cinzel generously | Use Cinzel for body copy |
| Keep body copy line height at 1.6–1.7 | Mix more than two fonts on one screen |
| Use Space Mono for the terminal sequence only | Use the wordmark font for anything except the brand name |

### Voice

| Do | Don't |
|----|-------|
| Set the scene before explaining the product | Lead with features or functionality |
| Let the user arrive at their own insight | Tell the user what to think or feel |
| Use present-tense, grounded language | Use spiritual jargon or faux-archaic language |
| Write in the avatar's distinct voice for their content | Use a generic brand voice across all four avatars |
