Release Notes

A running log of what's new — features, content, and fixes, newest first.

April 2026

Apr 6, 2026 Fix

Wordmark font name corrected in typography tokens

The Remachine Script font name in src/theme/typography.ts was updated to match the asset file exactly, preventing silent fallback to the system font on web. No visual change for users — this is a developer-facing correctness fix that ensures font loading is reliable across Expo native and web targets.

Apr 6, 2026 Content

Minor Arcana Midjourney prompt libraries added

All four suits of the Minor Arcana — Wands, Cups, Swords, and Pentacles — now have complete Midjourney prompt libraries (tasks #81–84). Each of the 56 Minor Arcana cards has a dedicated visual prompt for AI image generation, available in src/features/daily-draw/minorArcanaPrompts.ts. This brings full 78-card coverage to the daily draw feature's illustration pipeline.

Apr 6, 2026 Content

Major Arcana personality and soul card readings

Added interpretations for all 22 Major Arcana cards covering both the personality card (birth-month archetype) and soul card (life-path archetype) roles (task #44). Readings are surfaced during the onboarding flow after a user enters their date of birth, and live in src/features/onboarding/cardInterpretations.ts.

Apr 5, 2026 Feature

Remachine Script wordmark font integrated

Remachine Script is now registered as Majestic's official wordmark typeface, loaded through Expo's font system in TTF, EOT, and WOFF2 formats (task #79). It is available as the wordmark font family in typography tokens and renders on the splash screen, onboarding header, and any surface displaying the full app logo.

Apr 5, 2026 Launch

Majestic v1 — initial release

The first version of Majestic is live. Built on Expo Router with file-based navigation, Zustand state management, and a Supabase backend, the app ships with an 11-screen onboarding flow, four selectable avatar companions (Casper, Eli, Olivia, and Destiny), and a birth-card numerology engine that derives personality and soul cards from a user's date of birth. Authentication, profile creation, and the daily draw reading flow are all available at launch.
